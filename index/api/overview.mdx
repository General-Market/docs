# API Endpoints Index

The General Market API provides read access to real-time price data, Index Tracking Product (ITP) pricing, portfolio tracking, backtesting simulation, and lending position data.

## Base URL

In production, the API is proxied through Next.js rewrites at:

```text
https://generalmarket.io/api
```

The underlying data-node runs on port 8200. All paths documented in the API reference are relative to the base URL above.

## Response Format

All endpoints return JSON. A successful response contains the requested data directly in the response body:

```json
{
  "asset": "BTC",
  "price": 97432.15,
  "timestamp": 1708600000
}
```

Error responses return an appropriate HTTP status code with a JSON body:

```json
{
  "error": "Asset not found"
}
```

## Authentication

No authentication is required for read endpoints. The API is publicly accessible.

## Rate Limiting

No strict rate limits are enforced, but excessive request volume may result in temporary throttling.

## Quick Start

Fetch the latest price for Bitcoin:

<CodeGroup>
```bash cURL
curl https://generalmarket.io/api/price?asset=BTC
```

```javascript JavaScript
const res = await fetch("https://generalmarket.io/api/price?asset=BTC");
const data = await res.json();
console.log(data.price);
```

```python Python
import requests

resp = requests.get("https://generalmarket.io/api/price?asset=BTC")
data = resp.json()
print(data["price"])
```
</CodeGroup>

  "asset": "BTC",
